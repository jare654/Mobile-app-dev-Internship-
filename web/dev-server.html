🔒 Chrome Security Issue Fixed

✅ Solution: Development Server

Chrome blocks file:// URLs from making API calls. Use this development server to bypass the restriction:

# Step 1: Start Development Server
python3 -m http.server 8000

# Step 2: Open in Chrome
http://localhost:8000

# Step 3: Click "Advanced" → "Proceed to localhost" if security warning appears
🚀 Open Development Server

⚠️ Important Notes

  • Keep this server running while using the Flutter app
  • Only for development - not for production
  • Alternative: Use Firefox instead of Chrome
  • Alternative: Run Chrome with --disable-web-security

🔧 Quick Commands

# Start server in background
python3 -m http.server 8000 &

# Kill server when done
lsof -ti:8000 | xargs kill -9

# Chrome with disabled security (alternative)
chrome --disable-web-security --user-data-dir=/tmp/chrome_dev
